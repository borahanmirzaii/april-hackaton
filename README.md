# Active Hackathons Dashboard

A dark-themed HTML dashboard listing active hackathons from [DoraHacks](https://dorahacks.io/hackathon), organized by deadline urgency.

## Features

- **21 hackathons** sourced from DoraHacks
- Three urgency tiers: Closing Soon, 1-2 Weeks, 1-3 Months
- Clickable cards linking directly to each hackathon page
- Responsive grid layout
- Dark theme UI

## Usage

Open `index.html` in any browser.

## Data Source

All hackathon data sourced from [dorahacks.io/hackathon](https://dorahacks.io/hackathon) as of April 3, 2026.
